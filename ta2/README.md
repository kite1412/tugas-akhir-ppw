## Deskripsi Proyek

Portfolio ini dibuat menggunakan HTML dan CSS untuk memberikan tampilan modern dan responsif. Website ini terdiri dari beberapa bagian utama:

- **Home:** Menampilkan nama dan profesi saya.  
- **Tentang Saya:** Profil singkat, spesialisasi, pendidikan, dan lokasi saya.  
- **Keahlian:** Menampilkan kemampuan saya di Frontend, Backend, dan Mobile Development.  
- **Proyek Terbaru:** Showcase proyek-proyek yang sudah saya buat, lengkap dengan tautan ke GitHub.  
- **Hubungi Saya:** Form kontak untuk berkomunikasi langsung dengan saya.  

Website ini cocok untuk menunjukkan kemampuan saya dalam mengembangkan website dan aplikasi mobile.

## Fitur

- Navigasi mudah ke setiap bagian halaman  
- Responsif di desktop dan mobile  
- Showcase proyek dengan tautan GitHub  
- Form kontak untuk komunikasi langsung  
- Tampilan modern dengan efek hover dan shadow
